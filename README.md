## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `andleeb-seo-portfolio`)
2. Upload all three files (`index.html`, `style.css`, `script.js`)
3. Go to **Settings → Pages**
4. Set source to **main branch, root folder**
5. Your site will be live at `https://yourusername.github.io/andleeb-seo-portfolio/`

## Custom Domain

To use `andleebseo.com`:
1. Add a `CNAME` file to the repo containing: `andleebseo.com`
2. Update your domain DNS with a CNAME pointing to `yourusername.github.io`

## Contact Form (Production)

The form currently shows a demo success message. For a real form:
- **Formspree**: Replace `submitForm()` in `script.js` with a Formspree POST
- **Netlify Forms**: Add `netlify` attribute to the `<form>` tag
- **EmailJS**: Add EmailJS SDK and connect to your Gmail
